# instashop

__Описание:__ демо интернет-магазин


## Requirements

- PHP 8.4
- MySQL 8
- Composer
- Node.js



## Deploy

``
composer run setup
``



## Run

``
composer run dev
``


## Админка

[/k2/admin](https://instashop.nixtone.ru/k2/admin)
